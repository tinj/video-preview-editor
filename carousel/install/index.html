Famo.us Carousel

Installation

Follow these steps to install the Famo.us Carousel, explained using the Carousel:

Step 1:

Inside of your HTML, add the FamousSlider.js.

Step 2:

Add the following line to your HTML file 

<script type="text/javascript" src="/js/famousCarousel.js"></script>

Step 3:

In your javascript code, instantiate the Carousel:

FamousCarousel('#someDiv');

Customizing:

To customize the carousel, pass in options as the second argument:

FamousCarousel('#someDiv', { 
  loop: true 
});
